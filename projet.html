Projets

🏛️ Projet « Dauphinéo » – Patrimoine Culturel

Ce projet valorise le patrimoine culturel de la région Dauphiné. Mon équipe de 7 personnes et moi avons choisi de mettre en avant les Sites Patrimoniaux Remarquables (SPR).

Nous avons intégré une fonctionnalité de guide de visite, une base de données relationnelle, et une interface web. Le projet a nécessité une forte gestion de projet, de la modélisation et du travail collaboratif.

Nous avons utilisé Laravel pour le back-end, React pour le front-end, et j'ai activement participé aux deux parties ainsi qu’à la conception de la base de données.

🧭 Application de gestion d’événements d’entreprise

Ce projet consistait à développer une application de gestion d’événements pour les entreprises (team building, afterwork…).

Nous avons mis en œuvre une architecture MVC avec JavaFX, en réalisant l’ensemble des étapes de modélisation, conception et développement d'une application desktop complète.

📊 Analyse de données nutritionnelles et Nutri-Score

À partir de données issues de l’OpenFoodFacts, nous avons produit des graphiques analytiques pour mettre en évidence les corrélations entre les variables nutritionnelles et le Nutri-Score.

Le tout a été rédigé en anglais dans une démarche scientifique rigoureuse.
