Le projet

En équipe de 7, nous avons conçu et développé une application web destinée à promouvoir le patrimoine français. Pour ce faire, nous avons réalisé une carte interactive avec un aspect communautaire afin de faire connaître les maisons d'illustres françaises.

Ce projet s'est déroulé en trois phases sur plusieurs mois : gestion de projet, conception et développement.

Phase de gestion de projet

Pour cette première phase, l'objectif était de nous préparer pour la suite du projet.

  • Définition des besoins fonctionnels et construction du diagramme de cas d'utilisation
  • Définition et mise en page des personas
  • Réalisation des premières maquettes en fil de fer

Phase de conception

Pour cette deuxième phase, l'objectif était de concevoir l'application avant de pouvoir commencer le développement.

  • Création des maquettes détaillées
  • Élaboration des diagrammes de classes pour définir les relations entre les différents utilisateurs
  • Conception du schéma pour la base de données

Phase de développement

Pour cette dernière phase, l'objectif était de développer l'application.

  • Développement du backend avec Laravel
  • Création du frontend en React
  • Intégration d'une carte interactive
  • Rédaction du rapport final du projet
